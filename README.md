## Install Instructions

 > Install requires Neovim 0.9+. Always review the code before installing a configuration.

Clone the repository and install the plugins:

```sh
git clone git@github.com:rguruprakash/nvim_config ~/.config/rguruprakash/nvim_config
NVIM_APPNAME=rguruprakash/nvim_config/ nvim --headless +"Lazy! sync" +qa
```

Open Neovim with this config:

```sh
NVIM_APPNAME=rguruprakash/nvim_config/ nvim
```
